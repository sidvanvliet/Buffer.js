Buffer.js demo

You can start typing to try it out.
The callback function will be called after 1 second (buffer-js-delay) of no events in the input box.

A short breakdown of the code:
<input type="text" name="username" buffer-js buffer-js-delay="1" buffer-js-callback="callbackFunc">
buffer-js this initiates Buffer.js on said input.
buffer-js-delay="1" this sets the buffer (time that needs to expire before the callback is called) to 1 second
buffer-js-callback="callbackFunc" this tells Buffer.js that it needs to call function callbackFunc() after the buffer expires.
